# Texture attributions

- `milkyway_gaia_4k.jpg` — Real all-sky image of the night sky in galactic
  coordinates (equirectangular), assembled from ESA Gaia EDR3 measurements of
  ~1.8 billion stars. Credit: ESA/Gaia/DPAC, CC BY-SA 3.0 IGO
  (acknowledgement: A. Moitinho). Obtained via the MIT-licensed
  [mw_plot](https://pypi.org/project/mw-plot/) package
  (`MW_edgeon_edr3_unannotate.jpg`), then background-pedestal subtracted
  (−16/255, gamma 1.18) so dark sky composites to black under additive
  blending, and downscaled from 6500x3250 to 4096x2048 for web delivery.

- `moon_2k.jpg` — Full-color mosaic of the lunar near side, imaged by NASA's
  Lunar Reconnaissance Orbiter (NASA imagery, public domain). Obtained via the
  MIT-licensed [CoryG89/MoonDemo](https://github.com/CoryG89/MoonDemo)
  repository (`img/maps/moon.jpg`) and downscaled from 4096x2048 to 2048x1024
  for web delivery.
